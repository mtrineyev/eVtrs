# Курс по основам Python

Цей курс містить приклади програм на Python

## Копіювання коду з GitHub
```bash
git clone git@github.com:mtrineyev/eVtrs.git
cd eVtrs
```

## Створення віртуального середовища
Linux/Mac

```bash
python3 -m venv .venv
source ./.venv/bin/activate
pip install -r ./requirements.txt
```

Windows
```bash
py -m venv .venv
.\.venv\bin\activate
pip install -r requirements.txt
```

## Виконання першого скрипта
```bash
python ./src/essential/example_1.py
```
